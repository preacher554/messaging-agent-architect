# Business Use Cases

Reusable business-layer packages for managed messaging agents.

- AI Receptionist — answers common inquiries and routes to human admin.
- AI Sales Receptionist — qualifies leads, captures needs, and prepares handoff summaries.
- Support Assistant — triages support requests and gathers context.
- Booking Assistant — coordinates appointment intent and handoff/booking tools.
- Ecommerce/Catalog Assistant — answers catalog questions when a product data adapter is enabled.
