# Package Structure v1

Messaging Agent v1 only supports Basic and Pro as standard packages.

## Package 1 — Basic: AI Receptionist

### Positioning

AI receptionist for messaging channel that answers common end user questions quickly and consistently.

### Best for

- small local services,
- salons/barbershops,
- laundry,
- clinics with simple FAQ,
- courses/classes,
- repair/service businesses,
- owner-operated SMBs.

### Core value

end user chats stay answered even when the owner/admin is busy.

### AI can

- greet customers,
- answer FAQ,
- explain services,
- share opening hours/address,
- explain basic pricing if approved,
- explain order/booking steps,
- collect name and simple needs,
- remember basic end user context within tenant memory,
- hand off to admin.

### AI cannot

- qualify complex leads,
- run advanced sales strategy,
- promise discounts,
- close final custom deals,
- check SKU/stock,
- take payment decisions,
- handle complaints requiring authority,
- integrate with CRM/payment/ads by default.

### Required end user data

- business name,
- messaging channel account,
- business description,
- service list,
- FAQ,
- approved basic prices,
- opening hours,
- address/service area,
- admin handoff contact,
- tone preferences.

## Package 2 — Pro: AI Sales Receptionist

### Positioning

AI sales receptionist for messaging channel that answers customers, understands needs, qualifies leads, and prepares handoff summaries for human follow-up.

### Best for

- service businesses,
- agencies,
- clinics/beauty services,
- courses/education,
- travel/services,
- B2B SMBs,
- high-inquiry businesses where admin follow-up matters.

### Core value

Incoming chats become cleaner, more qualified leads instead of scattered conversations.

### AI can

Everything in Basic, plus:

- ask discovery/probing questions,
- identify end user intent,
- classify lead as hot/warm/cold,
- capture pain point, urgency, budget signal, and desired outcome,
- answer basic objections from approved scripts,
- route customers to the right offer/service,
- perform light follow-up according to approved rules,
- prepare lead summary for admin.

### AI cannot

- finalize custom pricing,
- promise discounts,
- sign contracts,
- issue refunds,
- provide regulated advice,
- guarantee results,
- auto-close deals without human policy,
- handle SKU/stock unless Commerce add-on is explicitly enabled.

### Required end user data

Everything in Basic, plus:

- offer/service packages,
- ideal end user profile,
- qualification questions,
- objection-handling script,
- lead status rules,
- handoff thresholds,
- follow-up rules.
